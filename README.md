# SmallPDF-like Editor

一款基于 PySide6 和 PyMuPDF 的轻量级 PDF / 图片页面编排工具，可像 Smallpdf 一样拖拽排序、设置页面尺寸与压缩参数，并导出为高质量的 PDF 文件。

---

## ✨ 功能特点

- **多格式导入**  
  支持 PDF、PNG、JPG、JPEG、WEBP、BMP、TIFF 文件，可直接拖拽或通过菜单导入。
- **卡片式预览**  
  每个页面以卡片形式展示缩略图、文件名和页码，选中高亮，一目了然。
- **拖拽排序**  
  支持内部卡片拖拽调整顺序，也可从外部拖入新文件。
- **旋转与删除**  
  选中卡片后可旋转 90° 或删除，快捷键 `R` 和 `Delete` 快速操作。
- **灵活的导出设置**  
  - 自定义页面尺寸（预设 A4、A3、Letter 等）或使用素材原始尺寸。  
  - 三种压缩模式：高清保留矢量、推荐压缩（矢量+图片压缩）、强压缩（全部栅格化）。  
  - 可调节 DPI、JPEG 质量、灰度转换、PDF 结构优化。
- **导出为 PDF**  
  按照当前顺序和设置导出为一份 PDF 文件，支持边距控制。

## 🖥️ 系统要求

- Python 3.8 或更高版本
- 支持的操作系统：Windows / macOS / Linux（具备 Qt 环境）

---

## 📦 依赖安装

项目依赖以下 Python 库：

- `PySide6` – Qt 界面框架
- `pymupdf`（即 `fitz`）– PDF 处理与渲染

安装命令：

```bash
pip install PySide6 pymupdf
```

> 如果 `pymupdf` 安装失败，可尝试 `pip install PyMuPDF`（两者等价）。

---

## 🚀 运行程序

将代码保存为 `main.py`，然后在终端中执行：

```bash
python main.py
```

## 🧭 使用指南

### 1. 导入文件

- **菜单**：点击工具栏 **“导入文件”**，选择 PDF 或图片。
- **拖拽**：将文件或文件夹直接拖入主窗口（支持递归遍历文件夹）。
- 导入后，每个页面会生成一个卡片，并显示缩略图。

### 2. 管理页面

- **选择**：点击卡片即可选中（蓝色边框高亮）。
- **排序**：长按卡片并拖动到目标位置，松开即可移动。
- **旋转**：选中卡片后，点击工具栏 **“选中右转 90°”** 或按 `R` 键，旋转当前卡片。
- **删除**：选中卡片后，点击 **“删除选中”** 或按 `Delete` 键移除。
- **清空**：点击 **“清空全部”** 移除所有页面（会二次确认）。

### 3. 设置导出参数

点击工具栏 **“页面/压缩参数”** 打开设置对话框，包含两大部分：

#### 页面尺寸
- **固定尺寸**：从预设（A4、A3、Letter 等）中选择，或手动输入宽高（单位 mm）。
- **原始尺寸**：勾选后，每个页面使用素材自身尺寸（PDF 使用页面大小，图片按导入 DPI 换算）。
- **边距**：设置页面四周边距（单位 mm）。

#### 压缩参数
- **压缩模式**：
  - **高清/保留矢量**：PDF 页面保留矢量，图片原图插入（无压缩）。
  - **推荐压缩**：PDF 保留矢量，图片按 DPI 和 JPEG 质量压缩（平衡体积与质量）。
  - **强压缩**：所有内容（包括 PDF 文字）栅格化为图片，体积最小但文字不可搜索/复制。
- **图片/栅格化 DPI**：影响图片质量和分辨率。
- **JPEG 质量**：1～95，数值越高质量越好，文件越大。
- **灰度**：将所有页面转为灰度。
- **保存时清理和压缩 PDF 结构**：启用后对输出 PDF 进行清理和压缩（`garbage=4`, `clean=True`）。

### 4. 导出 PDF

点击 **“导出 PDF”**，选择保存路径（自动添加 `.pdf` 后缀）。  
导出过程中会显示进度条，完成后弹出提示；若有错误（如某页读取失败）会给出警告列表。

## ⌨️ 快捷键

| 按键       | 操作               |
|------------|--------------------|
| `Delete`   | 删除当前选中的卡片 |
| `R`        | 将选中卡片右转 90° |

---

## ⚙️ 高级说明

- **缓存机制**：PDF 和缩略图会缓存以提升性能，关闭程序时自动释放。
- **图片渲染**：缩略图采用双线性缩放，导出时按设定 DPI 重采样。
- **多文件夹导入**：拖入文件夹会递归扫描内部所有支持格式的文件（按路径排序）。

---

## ⚠️ 注意事项

- 导出大文件（数百页）时，请确保有足够内存。
- “强压缩”模式会将页面转为 JPEG 图片，文字将不可选中，仅适用于预览或非文本场景。
- 旋转操作会修改卡片的旋转属性，不影响源文件。
- 若导入的图片包含透明通道，导出时会自动填充白色背景。
